# 개념 시각화 방법론 (Concept Visualization)

## 핵심 원리: Conceptual Metaphor Theory

추상적 개념(타겟 도메인)을 구체적이고 익숙한 오브젝트(소스 도메인)를 통해 시각화한다.

### 3가지 메타포 유형

| 유형 | 정의 | 예시 |
|------|------|------|
| 구조적 | 한 개념을 다른 개념의 구조로 이해 | "정밀함 = 그리드 시스템" |
| 방향적 | 공간적 방향으로 개념 조직 | "시작 = 원점 좌표" |
| 존재론적 | 추상 경험을 물리적 실체로 | "절제 = 비어있는 공간" |

## 8가지 시각적 메타포 디자인 원칙

1. **관련성**: DRAFT. 제품을 1차 오브젝트로 사용하여 브랜드와 직접 연결
2. **명확성**: 설명 없이도 즉시 인식 가능한 메타포
3. **친숙함**: 대상 사용자(성인 남성 전문직)가 공감하는 이미지
4. **단순성**: 복잡한 개념, 단순한 시각적 실행 — 하나의 지배적 메타포
5. **일관성**: 전체 시리즈에서 동일한 Blueprint 스타일과 어노테이션 시스템
6. **혁신성**: 디테일 콜아웃이 구체(제품)와 추상(가치)을 연결하는 브릿지 역할
7. **감정적 연결**: 씬이 해당 개념의 느낌을 환기
8. **확장성**: 전체 페이지와 썸네일 모두에서 작동

## DRAFT. Core Value 메타포 매핑

| 가치 | 소스 도메인 (구체적) | 타겟 도메인 (추상적) | 1차 오브젝트 | 디테일 콜아웃 |
|------|-------------------|-------------------|-------------|-------------|
| PRECISION (정밀함) | 그리드 노트북 + 자 | 수학적 정확성 | 등각투영 노트북 페이지 + 5mm 그리드 | 그리드 교차점 확대 → 90° 표시 |
| ORIGIN (시작점) | 펼친 노트북 + 첫 획 | 시작, 기원 | 등각투영 펼친 노트북 | 펜촉-종이 접점 확대 → (0,0) 좌표 |
| RESTRAINT (절제) | 광활한 매트 + 펜 1자루 | 미니멀리즘, 여백 | 등각투영 데스크 매트 (대부분 비어있음) | 펜 확대 → 정밀 구조 |
| CRAFT (도구에 대한 존중) | 만년필 닙 단면도 | 숙련, 메커니즘 | 닙 단면 + 잉크 채널 | 닙 끝 확대 → 잉크 전달 순간 |

## 디테일 콜아웃 전략

**디테일 콜아웃 = 구체 ↔ 추상 브릿지**

- 메인 씬은 구체적 제품/장면 (소스 도메인)
- 콜아웃 확대 영역은 추상적 개념을 드러냄 (타겟 도메인)
- 예: PRECISION — 메인 씬은 노트북이지만, 확대하면 완벽한 90° 교차점이 보임
- 예: CRAFT — 메인 씬은 닙이지만, 확대하면 잉크가 종이에 전달되는 0.3초의 순간

## 씬 구성 원칙 (개념 일러스트)

### 필수 요소
1. **등각투영 1차 오브젝트**: DRAFT. 제품 중 하나 (제품 일러스트와 동일 수준 디테일)
2. **디테일 콜아웃**: 대시 원 → 리더라인 → 확대 인셋
3. **어노테이션 3개 이상**: 제품 일러스트와 동일한 리더라인 + 도트 + 2단 라벨
4. **치수선 1개 이상**: 개념적 측정값도 포함 가능 (예: "NEGATIVE SPACE: 94%")
5. **Ghost 엘리먼트**: 맥락 제공용 반투명 보조 요소

### 레이어 구성 순서
1. 1차 오브젝트 (등각투영, Primary 선 굵기)
2. 2차 오브젝트/디테일 (Secondary 선 굵기)
3. 디테일 콜아웃 (소스 원 + 리더 + 확대 인셋)
4. Ghost 엘리먼트 (opacity 0.15-0.35)
5. 어노테이션 (Annotation 선 굵기)
6. 치수선
7. 타이틀
